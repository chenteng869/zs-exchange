#### 集成验证 PoC 流程

```
PoC 1: OpenClaw → n8n
  1. OpenClaw 智能体收到用户消息
  2. 通过 Function Calling 调用 n8n webhook
  3. n8n 工作流执行简单任务（如：HTTP 请求）
  4. n8n 返回结果给 OpenClaw
  5. OpenClaw 整理后回复用户

验收: 完整链路跑通 < 5 秒

PoC 2: n8n → 区块链存证
  1. n8n 工作流触发"存证"动作
  2. 调用存证网关 API
  3. 存证网关返回凭证（< 1 秒）
  4. 异步上链（< 5 秒）

验收: 凭证可链上验证
```

interface VerifyResponse {

  cert: NotarizeResponse;
