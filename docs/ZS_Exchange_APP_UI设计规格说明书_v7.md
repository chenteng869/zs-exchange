## 一、设计理念与视觉定位

### 1.1 核心关键词

```
极光尊享 (Aurora Premium) — 深邃、通透、科技感、金融级
```

### 1.2 视觉特征

| 特征 | 描述 |
|------|------|
| **背景** | 不再是纯黑 `#0B1124`，升级为**皇家深蓝** `#0F1B3D` + 多层极光渐变网格 |
| **卡片** | 玻璃拟态 2.0：`backdrop-filter: blur(24px) saturate(180%)`，透明度 55%-70% |
| **色彩** | 极光五色呼吸体系：金 / 青 / 紫 / 翠 / 粉，营造多色光晕效果 |
| **质感** | 每张卡片顶部有「高光线」装饰（`border-image` 渐变），金属感边框 |
| **动效** | 「光流体」动画（aurora-flow）— 背景光斑缓慢呼吸移动 |

### 1.3 三大主题系统

| 系统 | 适用场景 | 背景色调 |
|------|---------|---------|
| **Aurora Premium（极光尊享）** | 官网 / H5移动端 / User PC前台 | 深色 `#0F1B3D` 底 + 极光渐变 |
| **Dark Trading（深色专业终端）** | 交易页 / K线终端 | 更深的交易专用底色 |
| **Light Admin（浅色后台）** | 管理后台 / 风控 | 浅灰 `#F5F7FA` + 深色侧栏 |

## 十、动效与过渡

### 10.1 动画时长

| 时长 | 值 | 用途 |
|------|-----|------|
| **快速(fast)** | `150ms` | hover、focus状态切换 |
| **正常(normal)** | `250ms` | 展开/收起、淡入淡出 |
| **慢速(slow)** | `400ms` | 页面切换、复杂动画 |
| **极慢(slower)** | `600ms` | 大型转场 |
| **极光呼吸(aurora)** | `12-14s` | 背景光斑循环 |
| **极光漂移(drift)** | `18-22s` | 背景光斑慢漂移 |

### 10.2 缓动函数

| 函数 | 值 | 用途 |
|------|-----|------|
| **default** | `cubic-bezier(0.4, 0, 0.2, 1)` | 标准过渡 |
| **ease-in** | `cubic-bezier(0.4, 0, 1, 1)` | 进入动画 |
| **ease-out** | `cubic-bezier(0, 0, 0.2, 1)` | 退出动画 |
| **bounce** | `cubic-bezier(0.68, -0.55, 0.265, 1.55)` | 弹性反馈 |
| **aurora** | `cubic-bezier(0.45, 0, 0.55, 1)` | 极光专用（平滑往返） |

### 10.3 极光动画（v7 核心）

| 动画名 | 周期 | 效果描述 |
|--------|------|---------|
| **aurora-flow** | `12-14s` | 光斑：缩放(0.7→1.15→0.95) + 位移(0→+30,-20→-30,+20) + 透明度循环 |
| **aurora-drift** | `18s` | 光斑慢漂：缩放(0.6→1.2→1) + 位移(0→-50,+40→0) |
| **aurora-rise** | `22s` | 光斑上升：上下浮动(0→±20px) + 缩放微变 |
| **gold-shine** | `3s` | 金色脉冲：boxShadow在 24px↔48px 之间呼吸闪烁 |

### 10.4 过渡动画

| 触发 | 动效 | 时长 |
|------|------|------|
| 卡片hover | 上浮2px + 边框变色 + 阴影加深 | 300-400ms ease-out |
| 按钮hover | 上浮1px + 阴影扩散 | 200-300ms ease-out |
| 页面进入 | fadeInUp（透明+上移10px→可见+原位） | 400ms ease-out |
| 弹窗出现 | scaleIn（缩小95%→正常+淡入） | 300ms ease-out |
| Tab切换 | 下划线滑位 + 颜色渐变 | 250ms ease-out |
| 数字滚动 | 数字从0滚动到目标值（用于资产数字） | 500-800ms |

### 10.5 无障碍（减弱动效）

```css
@media (prefers-reduced-motion: reduce) {
  .anim-aurora-flow,
  .anim-aurora-drift,
  .anim-aurora-rise,
  .anim-gold-shine {
    animation: none;  /* 尊重用户偏好时关闭所有极光动画 */
  }
}
```
