## 二、岗位 JD（Job Description）

### JD-01: 架构师 / 技术负责人

**岗位名称**: 高级架构师（AI & 区块链方向）— ZS-AI-Platform
**汇报对象**: CTO
**下属**: 后端 / 前端 / 测试 / 运维（虚线）
**地点**: 上海 / 远程（每周 1 天现场）
**薪酬范围**: **30 ~ 40K × 13 ~ 16 薪**（年薪 39 ~ 64 万）

#### 岗位职责

1. 主导 ZS-AI-Platform 整体架构设计（4 层架构：接入层 / 引擎层 / 数据层 / 基础设施层）
2. 负责 OpenClaw + n8n + Flowable + Hyperledger Fabric 4 大引擎的集成方案
3. 主导技术选型决策（数据库 / 中间件 / AI 模型 / 区块链框架）
4. Code Review 终审（关键模块 100% 覆盖）
5. 解决 P0/P1 技术难题，每周技术分享 1 次
6. 制定技术规范（API 规范 / 数据库规范 / 安全规范）
7. 阶段验收（M1/M2/M3/M4）的技术签字

#### 任职要求

**必备**:
- 8 年以上互联网 / 区块链架构经验，3 年以上 AI 项目经验
- 主导过 3 个以上百万级用户系统架构
- 精通微服务（Spring Cloud / Dubbo / gRPC）
- 精通 K8s + 中间件（PostgreSQL / Redis / Kafka）
- 有 Hyperledger Fabric / 以太坊 / FISCO-BCOS 实际项目经验
- 有 OpenClaw / n8n / Airflow / Temporal 等工作流引擎经验
- 良好的技术领导力，能管理 5 人以上团队

**加分**:
- 在 GitHub / GitLab 有 100+ Star 项目
- 有技术专利 / 论文
- 有开源社区贡献

#### 面试流程（4 轮）

1. **HR 沟通**（30min）— 薪资 / 期望 / 背景
2. **技术初面**（60min）— 架构设计 / 案例分析
3. **技术终面**（90min）— CTO 面试 + 现场架构白板
4. **HR 终面 + Offer**（30min）
