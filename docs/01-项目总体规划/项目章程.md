## 三、项目范围（Scope）

### 3.1 ✅ In-Scope（必须交付）

#### A. 基础平台（4 大引擎 + 9 种中间件）

- **AI 引擎**: OpenClaw（智能体） + 通义千问 / GPT-5 / Claude / DeepSeek
- **工作流引擎**: n8n（Queue Mode，支持 50+ 并发）
- **BPM 引擎**: Flowable 7.x（多级审批）
- **区块链引擎**: Hyperledger Fabric 2.5（联盟链 + 国链数据联盟）
- **数据库**: PostgreSQL 16 / Redis 7 / MongoDB 7 / Kafka 3.x / MinIO / Milvus 2 / ClickHouse / InfluxDB 2 / Fabric
- **基础设施**: K8s 1.29（3 master + 10 worker = 13 节点） + 32 台 ECS

#### B. 业务模块（4 大业务）

1. **AI 智能获客**: 抖音 / 小红书 / TikTok / Instagram / YouTube / 微信 6 平台数据接入 + AI 评估
2. **区块链存证 + DID**: 4 种身份验证（手机 / 邮箱 / 指纹 / 人脸）
3. **Web 管理后台**: 40+ 页面（智能体 / 工作流 / BPM / 区块链 / 获客 / 系统）
4. **移动 App + 小程序**: iOS + Android + 微信小程序

#### C. 运维与监控

- Prometheus + Grafana（7 仪表板 + 50 告警）
- Loki + ELK（日志聚合 + 全文搜索）
- Jaeger / SkyWalking（全链路追踪）
- 7×24 On-Call 排班

### 3.2 ❌ Out-of-Scope（不在本次范围）

- ZS Exchange 既有模块（n8n / openclaw / bpm 等）UI 重构
- 公链（Ethereum / Solana）对接
- 自研大模型训练
- 硬件钱包（仅软钱包 + 手机 OTP）
- 第三方支付通道（Visa / Mastercard）
- 海外 KYC 监管牌照申请

### 3.3 🔄 待定项（Pending Decision）

- 智能合约法律效力（中国 / 新加坡两地）
- AI 内容生成的版权归属
- 数据联盟会员费分润模式

### 8.2 接受准则（Risk Acceptance Criteria）
