### 2.2 一键安装脚本（Ubuntu 22.04）

```bash
#!/bin/bash
# bootstrap.sh - 一键安装开发环境
# 使用：bash bootstrap.sh

set -e

echo "==> 1. 系统更新"
sudo apt update && sudo apt upgrade -y

echo "==> 2. 安装基础工具"
sudo apt install -y curl wget git vim nano htop \
  build-essential cmake pkg-config libssl-dev \
  zip unzip tree jq yq netcat-openbsd \
  software-properties-common ca-certificates gnupg lsb-release

echo "==> 3. 安装 Git (2.40+)"
sudo add-apt-repository ppa:git-core/ppa -y
sudo apt update
sudo apt install -y git

echo "==> 4. 安装 Docker (24+)"
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER
rm get-docker.sh

echo "==> 5. 安装 Docker Compose"
sudo apt install -y docker-compose-plugin

echo "==> 6. 安装 Node.js (20 LTS)"
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
sudo npm install -g pnpm@8 yarn npx

echo "==> 7. 安装 Python (3.11+)"
sudo add-apt-repository ppa:deadsnakes/ppa -y
sudo apt update
sudo apt install -y python3.11 python3.11-venv python3.11-dev python3-pip

echo "==> 8. 安装 Go (1.21+)"
GO_VERSION="1.21.5"
wget -q https://go.dev/dl/go${GO_VERSION}.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go${GO_VERSION}.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' | sudo tee -a /etc/profile
echo 'export GOPATH=$HOME/go' | sudo tee -a /etc/profile
echo 'export PATH=$PATH:$GOPATH/bin' | sudo tee -a /etc/profile
rm go${GO_VERSION}.linux-amd64.tar.gz

echo "==> 9. 安装 Java (17 LTS)"
sudo apt install -y openjdk-17-jdk maven

echo "==> 10. 安装 K8s 工具"
curl -LO "https://dl.k8s.io/release/v1.29.0/bin/linux/amd64/kubectl"
sudo install -o root -g root -m 0755 kubectl /usr/local/bin/kubectl
rm kubectl

curl -fsSL -o get_helm.sh https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3
sudo bash get_helm.sh
rm get_helm.sh

echo "==> 11. 安装 Terraform & Ansible"
sudo apt install -y ansible
TERRAFORM_VERSION="1.6.5"
wget -q https://releases.hashicorp.com/terraform/${TERRAFORM_VERSION}/terraform_${TERRAFORM_VERSION}_linux_amd64.zip
sudo apt install -y unzip
sudo unzip -o terraform_${TERRAFORM_VERSION}_linux_amd64.zip -d /usr/local/bin/
rm terraform_${TERRAFORM_VERSION}_linux_amd64.zip

echo "==> 12. 配置 Zsh + Oh-My-Zsh (可选)"
sudo apt install -y zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

echo ""
echo "✅ 全部安装完成！请重新登录或 source /etc/profile 加载环境变量"
echo "下一步：bash setup-git.sh 配置 Git"
```

### 2.3 macOS 一键安装

#!/bin/bash

### 2.4 Windows WSL2 一键安装

```powershell
# 在 PowerShell (管理员) 中执行

# 1. 启用 WSL2
wsl --install -d Ubuntu-22.04

# 2. 设置 WSL2 为默认
wsl --set-default-version 2

# 3. 安装 Windows Terminal (推荐)
winget install Microsoft.WindowsTerminal

# 4. 安装 Docker Desktop (勾选 WSL2 后端)
winget install Docker.DockerDesktop

# 5. 进入 WSL2
wsl

# 6. 在 WSL2 中执行 Ubuntu bootstrap.sh
curl -fsSL https://raw.githubusercontent.com/zs-exchange/dev-setup/main/bootstrap.sh | bash
```

### 5.2 SSH Key 生成

```bash
# 1. 检查是否已有 SSH Key
ls -la ~/.ssh/

# 2. 生成新的 SSH Key（使用企业邮箱）
ssh-keygen -t ed25519 -C "your.email@zs-exchange.com"

# 3. 启动 ssh-agent
eval "$(ssh-agent -s)"

# 4. 配置 SSH Config
cat >> ~/.ssh/config << 'EOF'
Host git.zs-exchange.com
  HostName git.zs-exchange.com
  Port 22
  User git
  IdentityFile ~/.ssh/id_ed25519
  IdentitiesOnly yes
EOF

# 5. 添加 SSH Key 到 ssh-agent
ssh-add ~/.ssh/id_ed25519

# 6. 复制公钥
cat ~/.ssh/id_ed25519.pub
# 复制输出，粘贴到 GitLab → Preferences → SSH Keys
```

## 八、常见问题（FAQ）

### Q1: Docker 启动报权限错误
```bash
sudo usermod -aG docker $USER
newgrp docker
```

### Q2: WSL2 中 Docker 速度慢
- 关闭 Windows Defender 对 WSL 的实时保护
- 项目代码放在 WSL 文件系统（`~/project`），不要放在 `/mnt/c/`

### Q3: 端口冲突
- 修改 `docker-compose.yml` 中对应服务的 `ports` 配置（左侧端口）
- 左侧 5432 → 5433，依此类推

### Q4: 内存不足
- Docker Desktop → Settings → Resources → Memory: 至少 8 GB
- 关闭不常用的服务（注释 docker-compose.yml 中的 services）

### Q5: SSH Key 失效
```bash
ssh-add -D   # 删除所有 keys
ssh-add ~/.ssh/id_ed25519   # 重新添加
```

### Q6: 提交被 Husky 拦截
- 检查 ESLint / Prettier 报错
- `npm run lint` 手动运行修复
