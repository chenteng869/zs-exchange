# ZS-AI-Platform 项目成本敏感性分析报告

**版本**：v1.0
**编制日期**：2026-06-11
**适用范围**：ZS-AI-Platform 一期建设
**分析方法**：单因素敏感性 + 多档预算方案对比 + 5 年 TCO 模型
**密级**：内部公开

---

## 一、执行摘要

本报告针对 ZS-AI-Platform 智能体协作与可信存证平台项目，从战略、业务、技术、财务四个维度建立 **3 档预算方案**（基础版 / 标准版 / 高级版），量化分析其在功能覆盖、性能指标、风险等级、投资回收等关键指标上的差异，并输出明确的预算推荐与决策依据。

### 核心结论

| 维度 | 基础版 | 标准版 ⭐推荐 | 高级版 |
|------|--------|--------|--------|
| **预算区间（万元）** | 50 – 78 | 157 – 238 | 320 – 450 |
| **中位值（万元）** | 64 | 197 | 385 |
| **功能覆盖度** | 60% | 100% | 100% + AI 增强 |
| **性能-并发智能体** | 20 | 50+ | 200 |
| **存证响应** | ≤5 秒 | ≤3 秒 | ≤1 秒 |
| **服务可用性 SLA** | 99.5%（4 小时停机/月） | 99.9%（45 分钟停机/月） | 99.99%（4.5 分钟停机/月） |
| **风险等级** | 🔴 高 | 🟡 中 | 🟢 低 |
| **投资回收期（月）** | 14 | 22 | 30 |
| **5 年 TCO（万元）** | 245 | 612 | 1,180 |
| **NPV@10%（5 年，万元）** | 142 | 386 | 712 |

**推荐方案**：**标准版**（预算 197 万元，区间 157-238 万元）。其综合得分 88.5/100，在功能完整性、性能达标度、风险可控性、投入产出比四项关键指标上取得最优平衡。基础版 60% 功能覆盖难以满足可信存证合规要求，高级版功能溢出严重（多语言/异地灾备对一期业务无紧迫价值），均不推荐。

## 五、5 年 TCO（总拥有成本）分析

### 5.1 TCO 假设
- 折现率：8%（金融业典型 WACC）
- 通胀率：3%/年（人力与云资源）
- 硬件折旧：3 年线性，残值率 5%
- 软件订阅：每年按 5% 涨价
- AI API：每年按 10% 下降（规模化效应）

### 5.2 5 年 TCO 现金流（万元）

| 年份 | 基础版 | 标准版 | 高级版 |
|------|--------|--------|--------|
| Y1（建设期） | 67.1 | 216.7 | 453.2 |
| Y2（运营期） | 38.0 | 95.0 | 165.0 |
| Y3 | 39.5 | 98.5 | 170.0 |
| Y4 | 41.0 | 102.0 | 175.0 |
| Y5 | 42.5 | 105.5 | 180.0 |

> 注：Y1 包含全部开发人力与硬件采购；Y2 起以运维、订阅、AI API 为主；高级版含异地灾备链路费用 18 万/年。

### 5.3 TCO 敏感性分析

针对标准版进行单因素敏感性分析（±20% 变动对 5 年 TCO 的影响）：

| 敏感因子 | 变动幅度 | TCO 变化 | 敏感系数 | 排名 |
|---------|---------|---------|---------|------|
| 开发人力单价 | ±20% | ±65 万 | 0.30 | 1 |
| 硬件单价 | ±20% | ±48 万 | 0.22 | 2 |
| AI API 单价 | ±20% | ±26 万 | 0.12 | 3 |
| 软件许可订阅 | ±20% | ±28 万 | 0.13 | 4 |
| 团队规模 | ±2 人 | ±50 万 | 0.23 | 2 |
| 项目周期 | ±4 周 | ±26 万 | 0.12 | 3 |

**结论**：
- **最敏感因子**：开发人力单价（敏感系数 0.30），即人均月成本从 4 万涨到 4.8 万，TCO 增 6.5%
- **次敏感因子**：团队规模与硬件单价
- **最不敏感**：AI API 单价（受规模化降价对冲）

### 5.4 TCO 风险情景

|------|---------|--------------|
| 乐观 | 需求稳定 + 团队不流失 + 一次性验收 | -15%（-32.5 万） |
| 基准 | 按计划交付 | 0 |

## 七、决策建议与优先级

### 7.1 决策建议总览

| 推荐等级 | 方案 | 适用条件 |
|---------|------|---------|
| ⭐⭐⭐⭐⭐ 强烈推荐 | **标准版**（197 万） | 默认推荐，覆盖 90%+ 业务场景 |
| ⭐⭐⭐ 备选 | 基础版（64 万）→3 个月内升级到标准版 | 仅当现金流极度紧张时 |
| ⭐⭐ 谨慎考虑 | 高级版（385 万） | 仅当客户/监管明确要求 99.99% SLA |
| ⭐ 不推荐 | 单独实施基础版且不升级 | 长期视角下不可持续 |

### 7.2 决策检查清单

- [ ] 监管机构是否明确要求 99.99% SLA？（是 → 高级版；否 → 标准版）
- [ ] 业务方是否能接受 60% 功能范围？（否 → 标准版）
- [ ] 预算是否可承担 200 万一次性投入？（否 → 基础版+分阶段；是 → 标准版）
