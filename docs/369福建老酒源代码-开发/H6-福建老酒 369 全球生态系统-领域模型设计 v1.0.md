# 0\. 文件定位

它的作用是把《福建老酒 369 全球生态白皮书》中的业务，拆成开发团队可以理解和实现的领域模型。



本系统不是一个简单商城，而是由多个业务域组成：



```Plain Text

税务域
风控域
审批审计域

```



每个领域都必须明确：



```Plain Text
1. 核心对象是什么
2. 对象有哪些状态
3. 对象之间什么关系
4. 哪些事件会触发变化
```
