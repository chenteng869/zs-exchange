---



## 13\.2 释放核心公式



用户理论释放：



```Plain Text
Release_calculated =
Release_pool_month × User_effective_power ÷ Network_effective_power
```



用户实际释放：



```Plain Text
Release_actual =
min(
  Release_calculated,
  User_monthly_cap,
  User_remaining_quota
)
```
