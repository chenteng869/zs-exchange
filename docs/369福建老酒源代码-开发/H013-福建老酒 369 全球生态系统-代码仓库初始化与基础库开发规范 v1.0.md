# 20\. Outbox 规范



每个服务必须有：



```Plain Text
outbox_events
```



字段：



```Plain Text
id
event_type
payload
status
retry_count
next_retry_at
created_at
published_at
```



状态：



```Plain Text
pending
published
failed
dead
```



业务写入时：



```Plain Text
同事务写业务表 + outbox_events
```



后台任务：



```Plain Text
扫描 pending
发送 Kafka
成功改 published
失败增加 retry_count
超过次数进入 dead
```
