# H3\-福建老酒 369 全球生态系统\-工业级开发执行规范 v1\.0

## 第 0 阶段 \- 第 5 阶段



---



# 总体说明



工业级系统不是“先做功能，再补风控”。



正确顺序是：



```Plain Text
先定架构
再定规则
再定账本
再定财务
再定风控
再做业务
```



所以第 0 到第 5 阶段是：



```Plain Text
第 0 阶段：总体架构与合规技术评审
第 1 阶段：基础平台底座
第 2 阶段：规则引擎与配置中心
第 3 阶段：资产账本与流水底座
第 4 阶段：财务分账与税务底座
第 5 阶段：风控与审计底座
```



这几个阶段要做到：



```Plain Text
规则不写死
资产不乱改
财务可对账
风控可拦截
后台可审计
链上可追踪
```



公式：



```Plain Text
tax_amount = paid_amount × tax_rate
gross_amount = paid_amount + tax_amount
```
