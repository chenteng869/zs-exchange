### 5.3 PowerShell 5.1 --no-verify 硬约束

```text
直接 git commit --no-verify 可能被解析为 --no + -verify
正确：git commit -F <msg-file> --no-verify
       1. commit message 写入临时文件
       2. git commit -F 文件 --no-verify
       3. --no-verify 后置
       4. 提交后删除临时文件
```
