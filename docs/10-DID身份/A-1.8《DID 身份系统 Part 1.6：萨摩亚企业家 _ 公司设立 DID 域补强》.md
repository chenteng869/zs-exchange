# 1\. 为什么必须单独成域



萨摩亚官网 APP / 公司设立企业家 DID 不是普通金融 DID，也不是普通电商 KYB。



它需要覆盖：



```Plain Text
个人创业者身份
企业设立申请身份
公司结构身份
董事 / 股东 / UBO 身份
注册地址和注册代理人
官方文件签署
官方审查状态
公司注册证书
持续合规状态
```



这里的 DID 更像：



```Plain Text
Government Service DID
  + Entrepreneur Identity
  + Company Formation Credential
  + Corporate Compliance Credential
```



因此不能简单放到 `financial` 或 `commerce` 里，必须独立成：



```TypeScript
'samoa_enterprise'
```

# 14\. Samoa Anchor Payload Builder



## `core/domains/samoa-enterprise/samoa-enterprise-anchor.service.ts`



```TypeScript
import { AnchorHashService } from '../../anchor/anchor-hash.service';

export interface SamoaEnterpriseAnchorPayload {
  domain: 'samoa_enterprise';

  credentialType:
    | 'SamoaEntrepreneurIdentity'
    | 'SamoaCompanyFormationApplication'
    | 'SamoaCompanyIncorporationCertificate'
    | 'SamoaCompanyGoodStanding'
    | 'SamoaRegisteredAgent'
    | 'SamoaUBODeclaration';

  credentialHash: string;

  subjectDidHash: string;
  issuerDidHash: string;

  companyNumberHash?: string;
  applicationIdHash?: string;

  schemaHash: string;

  issuedAt: number;
}

export class SamoaEnterpriseAnchorService {
  constructor(
    private readonly hash: AnchorHashService,
  ) {}

  buildPayload(input: {
    credentialType: SamoaEnterpriseAnchorPayload['credentialType'];
    credential: unknown;
    subjectDid: string;
    issuerDid: string;
    schema: unknown;
    companyNumber?: string;
    applicationId?: string;
    issuedAt: number;
  }): SamoaEnterpriseAnchorPayload {
    return {
      domain: 'samoa_enterprise',
      credentialType: input.credentialType,
      credentialHash: this.hash.hashJson(input.credential),
      subjectDidHash: this.hash.hashString(input.subjectDid),
      issuerDidHash: this.hash.hashString(input.issuerDid),
      companyNumberHash: input.companyNumber
        ? this.hash.hashString(input.companyNumber)
        : undefined,
      applicationIdHash: input.applicationId
        ? this.hash.hashString(input.applicationId)
        : undefined,
      schemaHash: this.hash.hashJson(input.schema),
      issuedAt: input.issuedAt,
    };
  }
}
```
