## 本次已验证 / 已修复（无需申请任何 Key，现在就是真实数据）

- **行情数据**：CoinGecko / OKX / DeFiLlama / alternative.me(恐慌指数) —— 29 个 `/api/v1/crypto/*` 接口全部真实数据，修复了 `trades` 接口的 OKX 字段解析 bug。
- **链上余额查询**：ETH / BSC 用免费公共 RPC 真实查询（修复了 3 个已失效的公共节点：cloudflare-eth.com / eth.llamarpc.com / rpc.ankr.com），TRON 用 TronGrid 免费接口（修复了 TRC20 余额查询用的已下线旧接口）。
- **App 合约行情数据管道**：新增播种脚本 `npm run market:seed`（8 个常见合约）+ 常驻同步 worker `npm run market:sync`（每 10s 同步 ticker/深度、每 30s 同步 K线），让 `/api/v1/perp/market`（App 实际调用的接口）从"空表"变成真实数据源。**生产环境需要把 `market:sync` 配置成 pm2/systemd 常驻进程**，否则数据会停止更新。

## 已知限制（非 Key 问题，是环境限制）

- Binance 现货 API 在当前开发环境所在网络被地区限制（HTTP 451），系统按设计自动切换到 OKX，不影响功能，但如果生产服务器部署在同样受限的地区，也会一直走 OKX 兜底——建议确认生产服务器出口 IP 所在地区。
