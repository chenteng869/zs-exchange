## 二、详细章节设计

### 第1章：项目概述与视觉定位（约2500字，3个表格）

#### 文档定位与适用端说明
- 文档版本与调研参考来源（Stripe 2026 / Coinbase Advanced / OKX Web3 / Kraken Pro / Bybit V5）
- 技术栈：Next.js 14 + React 18 + Tailwind CSS + TypeScript
- 适用端：H5移动端 / User PC前台 / 官网 / Light Admin管理后台

#### 核心设计理念
- 极光尊享（Aurora Premium）主题核心关键词：深邃、通透、科技感、金融级
- 五大视觉特征：皇家深蓝背景、玻璃拟态2.0卡片、极光五色呼吸体系、金属感边框、光流体动画

#### 三大主题系统
- Aurora Premium（极光尊享）：适用场景、背景色调说明
- Dark Trading（深色专业终端）：交易页与K线终端专用
- Light Admin（浅色后台）：管理后台与风控场景

### 第3章：字体、排版与间距系统（约3500字，4个表格）

#### 字体族配置
- 正文字体栈：Inter → PingFang SC → Microsoft YaHei → 系统默认
- 等宽字体栈：JetBrains Mono → Fira Code → DIN
- 数字字体栈：DIN → Roboto Mono → JetBrains Mono

#### 字号体系
- Web/H5端专用字号：Hero超大标题至管理后台KPI数字共15级
- Design Tokens标准字号：xs至4xl共8级（含px与rem对照）
- H5移动端特殊字号：总资产数字、价格数字、涨跌幅、Tab导航文字等10类

#### 字重与行高
- 四级字重：Normal(400) / Medium(500) / Semibold(600) / Bold(700)
- 四类行高：全局基础(1.6) / 标题紧凑(1.2) / 正文舒适(1.5-1.6) / 数字表格(1.0)

#### 间距系统
- 4px基准网格：0至24共13级间距Token
- H5移动端常用间距：页面边距、卡片内边距、列表间距、按钮内边距、安全区域等10类场景
