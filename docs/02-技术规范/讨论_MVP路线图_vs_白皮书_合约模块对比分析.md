### 4.2 核心公式已经验证

白皮书提供了经过设计的精确公式，包括：

```typescript
// 初始保证金
Initial Margin = Position Notional / Leverage

// 维持保证金
Maintenance Margin = Position Notional × Maintenance Margin Rate

// 风险率
Risk Rate = Equity / Maintenance Margin

// 未实现盈亏（多头）
Unrealized PnL = (Mark Price - Entry Price) × Position Qty

// 标记价格
Mark Price = Index Price + Premium Index

// 资金费率
Funding Rate = clamp(Premium Index + Interest Rate, Floor, Cap)

// 强平价格（多头）
Liquidation Price = Entry Price × (1 - (Margin - Maintenance Margin) / Position Notional)
```
