# ZS Exchange - 每日站会模板

## 📋 站会流程（15分钟）

### 1. 同步上次站会待办（5分钟）
- 逐一确认上次站会提出的阻塞项和待办任务
- 已完成的打勾，未完成的说明原因

### 2. 今日工作汇报（每人2分钟）
- **昨天完成了什么？**
- **今天计划做什么？**
- **有什么阻塞？**

### 3. 风险预警（3分钟）
- 识别可能影响进度的风险
- 讨论解决方案和责任人

### 4. 确认结论（2分钟）
- 总结本次站会的决策和行动项
- 确认下次站会时间
