## 一、交易模块

### 1.1 现货交易

**功能描述**：支持主流加密货币的现货买卖交易

**核心功能**：

| 功能 | 说明 |
|------|------|
| 限价单 | 设定价格挂单，成交后结算 |
| 市价单 | 按当前市场价格立即成交 |
| 止损限价单 | 触发价格到达后以限价挂单 |
| 止损市价单 | 触发价格到达后以市价成交 |
| 冰山订单 | 隐藏真实挂单量，仅显示部分 |
| 追踪止损 | 止损价随价格动态调整 |
| FOK | 立即全部成交或全部取消 |
| IOC | 立即成交剩余部分取消 |

**交易对**：支持 500+ 交易对，涵盖 BTC、ETH、USDT 等主流币种

**页面路径**：
- H5：`/h5/trade`
- Web：`/trade/spot`

---

### 1.2 合约交易

**功能描述**：永续合约和交割合约交易，支持最高 100 倍杠杆

**核心功能**：

| 功能 | 说明 |
|------|------|
| 永续合约 | 无到期日，资金费率机制 |
| 交割合约 | 定期交割，USDT 本位 / 币本位 |
| 逐仓模式 | 单个仓位独立保证金 |
| 全仓模式 | 账户所有可用余额作为保证金 |
| 止盈止损 | TP/SL 自动触发 |
| 计划委托 | 预设条件触发下单 |
| 跟踪委托 | 跟踪止损订单 |
| 平仓 | 市价平仓 / 限价平仓 |

**合约引擎**：`src/lib/perp/`
- `perp-engine.ts` - 合约交易核心逻辑
- `margin-calculator.ts` - 保证金计算
- `liquidation-engine.ts` - 强制平仓引擎
- `funding-engine.ts` - 资金费率计算
- `contract-registry.ts` - 合约配置管理

**杠杆范围**：1x - 100x

**页面路径**：
- H5：`/h5/trade/futures`
- Web：`/trade/futures`

### 1.3 杠杆交易

**功能描述**：保证金借贷交易，放大收益与风险

**核心功能**：

| 功能 | 说明 |
|------|------|
| 逐仓杠杆 | 单个交易对独立保证金 |
| 全仓杠杆 | 账户共享保证金 |
| 借贷 | 借入资金进行交易 |
| 还款 | 归还借入资金和利息 |
| 爆仓 | 保证金不足时强制平仓 |
| 利率 | 按小时计息，浮动利率 |

**页面路径**：`/trade/margin`

### 1.4 期权交易

**功能描述**：欧式/美式期权交易，支持看涨期权和看跌期权

**核心功能**：

| 功能 | 说明 |
|------|------|
| 看涨期权 (Call) | 到期时以行权价买入标的 |
| 看跌期权 (Put) | 到期时以行权价卖出标的 |
| 欧式期权 | 仅到期日可行权 |
| 美式期权 | 到期前任意时间可行权 |
| 期权链 | 展示不同行权价的期权合约 |
| Greeks | Delta/Gamma/Theta/Vega/Rho |
| 隐含波动率 (IV) | 市场预期波动率 |
| 自动行权 | 到期价内期权自动行权 |
| 现金结算 | 以现金差价结算 |
| 实物结算 | 交割标的资产 |

**BSM 定价模型**：Black-Scholes-Merton 公式计算期权理论价格

**期权引擎**：`src/lib/options/`
- `options-engine.ts` - 期权交易核心
- `bsm.ts` - Black-Scholes 定价
- `option-chain.ts` - 期权链生成
- `settlement-engine.ts` - 结算引擎

**页面路径**：
- H5：`/h5/trade/options`
- Web：`/trade/options`

### 1.7 跟单交易

**功能描述**：跟随 KOL 交易员自动复制其交易操作

**核心功能**：

| 功能 | 说明 |
|------|------|
| 固定金额 | 每笔跟单固定 USDT 金额 |
| 比例跟单 | 按 KOL 仓位比例跟单 |
| 缩放系数 | 直接乘以 KOL 数量 |

```
KOL A 邀请 KOL B → KOL B 是 A 的 L1
KOL B 邀请 KOL C → C 是 B 的 L1；C 同时是 A 的 L2
KOL C 邀请 user D → D 是 C 的 L1；D 是 B 的 L2；D 是 A 的 L3
```

**返佣比例**：

| Level | 返佣系数 |
|-------|---------|
| L1 | 30% |
| L2 | 10% |
| L3 | 5% |
